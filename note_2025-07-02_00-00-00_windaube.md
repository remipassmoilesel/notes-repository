# Installer Windaube aka Windows

- Pour accéder aux clés USB dans vbox: `$ sudo usermod -aG vboxusers $USER` puis se reconnecter
- Démarrer une machine virtuelle Windows
- Se connecter à Windows avec un compte administrateur.
- Télécharger `MediaCreationTool.exe` https://support.microsoft.com/en-us/windows/create-installation-media-for-windows-99a58364-8c02-206f-aa6f-40c3b507420d
- Créer un ISO ou bruler une clé directement
