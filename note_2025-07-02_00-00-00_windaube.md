# Installer Windaube aka Windows

- Pour accéder aux clés USB dans vbox: `$ sudo usermod -aG vboxusers $USER` puis se reconnecter
- Démarrer une machine virtuelle Windows
- Se connecter à Windows avec un compte administrateur.
- Télécharger `MediaCreationTool.exe` https://support.microsoft.com/en-us/windows/create-installation-media-for-windows-99a58364-8c02-206f-aa6f-40c3b507420d
- Créer un ISO ou bruler une clé directement
- Si pas de TPM 2.0:
  - Juste après le démarrage sur la clé faire Maj + F10 pour ouvrir un terminal
  - Taper `regedit`
  - Chercher `HKEY_LOCAL_MACHINE\SYSTEM\Setup`
  - Clic droit sur `Setup` => `Nouveau / Clé`
  - Nom de la clé: `LabConfig`
  - Clic droit sur `BypassTPMCheck` => `Nouveau / Dword 32bits`
  - Editer la valeur, mettre 1
  - Continuer l'installation
  - Source: https://lecrabeinfo.net/tutoriels/installer-windows-11-sans-tpm-2-0-ou-secure-boot-installation-propre/
